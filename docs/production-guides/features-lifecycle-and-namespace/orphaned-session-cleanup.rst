Orphaned Session Cleanup (`spec.workshops[].orphaned`)
======================================================

با پارامتر **orphaned** می‌توان سشن‌هایی که کاربر آنها را رها کرده است (مثلاً پنجره مرورگر را بسته)، بعد از مدت زمان کوتاه پاک کرد.

**نمونه yaml:**
.. code-block:: yaml

   spec:
     workshops:
       - name: public-workshop
         capacity: 80
         orphaned: 5m

**نکته:** اگر orphaned را خیلی کوتاه بگذاری، ممکن است حتی جلسات سالم هم ناخواسته حذف شود. پس با دقت تنظیم کن و سناریوی مصرف را در نظر بگیر.
